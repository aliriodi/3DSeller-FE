Si lo que queremos es listar los npm packages instalados de manera global:
npm list -g

Y para que sólo nos muestre el package sin los sub-packages:
npm list --depth=0
npm list -g --depth=0

De este modo sólo nos muestra los packages principales (sin profundidad)
